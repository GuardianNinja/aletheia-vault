# Backend

This directory will contain the backend services for Aletheia Vault.

Planned responsibilities:

- Authentication and identity.
- Invention registry APIs.
- Licensing and contracts handling.
- Dividend and payouts processing.
- Market orchestration (Investor, Franchise, Public Crypto).
- Integration with the JARVONDIS Reserve accounting logic.

Implementation details (language, frameworks, databases) are intentionally
left open at this stage and can be decided based on contributor expertise.
