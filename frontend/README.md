# Frontend

This directory will contain the web frontend for Aletheia Vault.

Planned features:

- Invention submission and management interface.
- Library browsing and search.
- License request and negotiation flows.
- Investor and Franchise Market dashboards.
- Public Crypto Market views (for users who opt-in).
- Reserve transparency dashboards (JARVONDIS Reserve).

Framework choice (e.g., React, Vue, Svelte) is open and can be decided
when implementation begins.
